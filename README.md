# TinyVDB, header-only C++03 (limited) OpenVDB library.

TinyVDB is header-only C++03 OpenVDB library. Not all features in OpenVDB are implemented.

## How to use

Simply copy `tinyvdb.h` to your project.

## Optional feature

### Blosc

```
$ git submodule update --init
```
